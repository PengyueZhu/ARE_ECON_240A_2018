Conclusion

We see that there is a very small bias when using heteroskedastic robust standard errors when the data is homoskedastic. However, we run into a serious problem when using conventional standard errors when the data is heteroskedastic as the bias will be very large.

Although controlling for heteroskedasticity significantly reduces the bias when heteroskadisticity is present, the estimates will still be biased relative to correctly estimating standard errors under homoskedasticity. This is seen by comparing the difference between the mean heteroskedastic standard error in table 2 and the unfeasible one (0.179) to the difference between the mean homoskedastic standard error in table 1 to the unfeasbile one (0.005).
